## Makefile
A Makefile automates compilation, recompiling only what changed.
It keeps build commands organized and consistent, especially in multi-file C projects.
It saves time and makes your project easier for others to build.
Read this for more informations ([ https://hackernoon.com/how-to-create-a-library-in-c-with-a-makefile ])
